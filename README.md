# 🚲 Sistema de Estoque para Bicicletaria

Sistema desenvolvido para gerenciamento de estoque de uma loja de bicicletas.

## Tecnologias

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL

## Funcionalidades

- Login
- Cadastro de produtos
- Entrada de estoque
- Saída de estoque
- Histórico de movimentações
- Busca de produtos
